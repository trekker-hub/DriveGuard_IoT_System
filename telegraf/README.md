# Telegraf Folder

This folder contains the Telegraf configuration for DriveGuard.

## Setup

1. **Download Telegraf**:
   - Go to: https://portal.influxdata.com/downloads/
   - Select: Telegraf → Your OS → AMD64
   - Download and extract

2. **Place telegraf.exe here**:
   - Copy `telegraf.exe` (or `telegraf` on Mac/Linux) to this folder

3. **Configuration**:
   - `telegraf_driveguard.conf` is auto-generated by `run.py`
   - No manual editing required

## File Structure After Setup

```
telegraf/
├── README.md                    ← This file
├── telegraf.exe                 ← You add this
└── telegraf_driveguard.conf     ← Auto-generated
```

## Manual Run (Optional)

If you want to run Telegraf manually:

```bash
./telegraf --config telegraf_driveguard.conf
```
